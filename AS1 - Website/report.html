Development Report

Reflection on my web development learning journey

My Experience

During this module I learned the fundamentals of web development using HTML and CSS. At the beginning, I had little experience creating websites, so understanding how to structure pages using semantic elements such as header, nav, main, section and footer was an important step in my learning.

One of the key skills I developed was using CSS to control layout and design. I used display:grid to create responsive layouts and organise content across different pages. This allowed me to maintain a consistent structure while also creating a different layout for the projects page, as required in the assignment.

I chose a modern colour scheme using blue, white and dark tones to create a clean and professional look. I also used a gradient hero section and card-based design for projects to improve the visual appearance. The font was selected to be simple and readable across both desktop and mobile devices.

A challenge I faced was making the website responsive. Initially, the layout only worked well on larger screens. By using media queries, I adapted the design for smaller devices and implemented a hamburger menu to improve navigation on mobile phones.

I also tested and validated my HTML and CSS using online validation tools to ensure that my code met web standards. This helped me identify and fix errors during development.

Overall, this project helped me understand how to design, build and debug a website. I have improved my confidence in using HTML and CSS and developed a better understanding of responsive design and user interface principles.
